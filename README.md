This project converts a Google Gemini API key into a functional, autonomous CLI agent. The agent can interact with you, understand natural language instructions, and execute terminal commands on your system to accomplish multi-step tasks.

## Features

- **Interactive Agent Prompt**: Bootstraps an agent of your choice with a custom name and Gemini model.
- **Autonomous Command Execution**: Detects special command execution blocks (`[CMD]...[/CMD]`) generated by the LLM and runs them directly in your shell.
- **Feedback Loop**: Feeds command execution output (`stdout`) and error logs (`stderr`) back into the agent's context window.
- **Multi-Step Execution**: Continuously prompts the agent after a command runs so it can perform follow-up actions or finalize the output.
- **History Management**: Automatically cleans up and manages chat history upon exit.

## How It Works

1. **Initialization**: On the first iteration, you specify the agent's name and the exact Gemini model to use.
2. **User Input**: You type a request or a task you want to perform (e.g., "Find all Python files in this repo").
3. **Agent Decision**: The Gemini model processes your prompt and can reply with normal text or command execution block:
   ```text
   [CMD]find . -name "*.py"[/CMD]
   ```
4. **Command Execution**: If a `[CMD]` block is detected, the CLI intercepts it, executes the command in your terminal shell, and intercepts the output.
5. **Continuation Loop**:
   - If successful, the output is formatted as `Success:\n<output>` and sent to the agent along with a `CONTINUATION_PROMPT`.
   - If an error occurs, the exit code and error details are sent instead.
   - The agent inspects the output and can choose to run another command (triggering another loop) or finalize its answer.

## Setup & Installation

1. **Clone the Repository**:
   ```bash
   git clone <repository_url>
   cd <repository_folder>
   ```

2. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your Gemini API credentials:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```

## Usage

To start the CLI agent:

```bash
python path/to/main.py
```

1. **Initialize the Agent**:
   When prompted, enter your agent's name and the exact model identifier separated by a single space.
   
   Example:
   ```text
   BOB gemini-1.5-flash
   ```

2. **Commanding the Agent**:
   Type any task or question. For example:
   ```text
   Identify the files in the current folder and read the README.md
   ```

3. **Exiting**:
   Type `q` or `quit` to safely close the session and clear the agent's history.
